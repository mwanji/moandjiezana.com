Work

Salary Tracker

A small app to help us manage our domestic employees' salaries. The data is stored in a spreadsheet on Google Docs to get sharing and data synchronisation for free. I had never used the Google Docs Spreadsheets API before. It has a few quirks.

I used SafeSax and WAX to read and write the Atom feeds needed to communicate with Google Docs. Both are excellent libraries that are efficient, easy to learn and pleasant to use.

On the UI side, I used a ViewPager for the first time. Otherwise, I left it unstyled, because this is a private app.
